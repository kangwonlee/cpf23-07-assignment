## ex06 min and max
* 실습파일 : ex06.py
* 이름이 `func06`인 함수를 다음과 같이 완성하시오
* 매개변수로 여러 숫자 (정수 또는 실수) 를 가지고 있는 list 또는 tuple 을 하나 받아 들이시오
* 함수의 반환값:<br>
`dict`형으로 'min' key와 최소값의 index, 'max' key와 최대값의 index 를 pair 로 반환
* 위 함수만 해당 py 파일에 제출하시오
